# NetWatcher 2.2.0

NetWatcher 2.2.0 expands the project from a basic ping logger into a broader connection-diagnostics suite.

## Highlights

- Jitter, packet loss and quality scoring on the live dashboard
- Ping, TCP and HTTP/HTTPS target checks
- Target Manager with add, edit, rename and remove actions
- 5-minute, 30-minute, 1-hour and 24-hour graph ranges
- Modern Outage History and ISP Evidence reports
- 1-day, 7-day and 30-day evidence ranges
- Configurable log retention
- Expanded tray quick menu
- Experimental local Access Mode with DNS-over-HTTPS and TLS first-flight fragmentation

## Access Mode note

Access Mode is not a VPN and does not install a packet driver. It is intended for browsers and applications that respect the Windows proxy setting. Compatibility depends on the network and blocking method. Use it only where permitted.
