# NetWatcher 2.2.2

This is a focused visual-polish release.

- Fixes the cramped Graph / 5 minutes / Outage History layout.
- Replaces the old native-looking main buttons with rounded, theme-aware buttons.
- Uses blue emphasis for primary actions, subtle blue accents for reports, and restrained red styling for Stop/Remove.
- Does not change the monitoring, reporting, tray, target, or Access Mode architecture.
