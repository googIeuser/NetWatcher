# NetWatcher 2.2.5

- Prevented toolbar buttons from colliding when the main window is resized.
- Added minimum safe sizes for the main window and Settings window.
- Reworked the main dashboard into responsive status, graph, event, and summary regions.
- Reworked Settings spacing so section headings, checkboxes, inputs, and footer buttons no longer overlap.
- Settings now redraws atomically during resize to avoid ghosting and duplicated controls.
- Increased action-button widths so complete labels remain visible without ellipses.
- Enabled the GitHub update setting in local and release builds by using the public NetWatcher repository as the default source.
