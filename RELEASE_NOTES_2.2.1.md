# NetWatcher 2.2.1

This update makes the main dashboard easier to understand for non-technical users.

- Each monitored connection now uses a compact three-line summary instead of a cramped technical table.
- Internet quality is shown with plain-language meanings such as “Excellent - stable connection”.
- The unexplained numeric quality score is hidden on the main screen and remains available in detailed statistics and reports.
- “Jitter” is presented as “Variation” on the dashboard.
- Gateway health is described separately as “Local network OK” or a local network problem.
- The bottom status bar now clearly shows Monitoring, Internet quality, Access Mode, samples, outages and downtime.
